## Version 2.1.0

**Released:** 2024-06-14

### New features

<Check>Use `e.preventDefault()` instead of css `pointer-events: none;` when dragging and touching.</Check>

<Check>Add options `disableDraggingClass` and `disableWheelingClass`.</Check>

<Check>Add options `dragReleaseOnBounds` and `wheelReleaseOnMinMax` to enable page scrolling in some cases.</Check>

<Check>Add methods `isOnBoundTop`, `isOnBoundBottom`, `isOnBoundLeft`, `isOnBoundRight`, `isOnBoundX`, `isOnBoundY`, `isOnMinScale`, `isOnMaxScale`.</Check>
