# Zoomist

Zoomist is a lightweight, zero-dependency TypeScript library that lets you add zoom, drag, and pinch-to-zoom interactions to any HTML element on your page. It works seamlessly on both desktop and mobile devices.

## Why Zoomist?

Zoomist focuses on simplicity and flexibility. Unlike heavier image viewer libraries, Zoomist works on **any HTML element** — not just images. It has zero dependencies, ships as both UMD and ES module builds, and gives you a comprehensive event API to hook into every interaction.

```typescript
import 'zoomist/css'
import Zoomist from 'zoomist'

const zoomist = new Zoomist('.zoomist-container', {
  maxScale: 4,
  bounds: true,
  slider: true,
  zoomer: true
})
```

## Browser support

Zoomist works in all modern browsers including Chrome, Firefox, Safari, and Edge. Mobile browsers with touch support are fully supported, including iOS Safari and Android Chrome.
